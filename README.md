\# Elderly Care AI



A comprehensive elderly care management platform with fall risk prediction, vitals tracking, and multilingual support.



\## Features

\- Resident Management

\- Vitals Tracking with Charts

\- AI-powered Fall Risk Prediction

\- Real-time Alerts System

\- Multilingual Support (English, Bengali, Arabic, French)



\## Tech Stack

\- Frontend: Vite + React + TypeScript + Tailwind CSS + Shadcn/ui

\- Backend: Express.js + Prisma + PostgreSQL (Railway)

\- Authentication: Clerk

\- Deployment: Vercel (Frontend), Railway (Backend)
